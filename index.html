Z-Tool

Support Internal Utility Tool

This utility tool was built by Tahur 😎 initially just for fun and for my own use. Later as created a barebone version for to use inside the team then improvised based on feedback.

  • Yes, you can totally do most of this in Excel, but this tool will save you at least 2 seconds (and likely a few sighs) each time!
  • Edge cases? Haven't tested much. If you break it or get a wrong result, please accept my heartfelt \_()_/.
  • Most major code chunks were generated by Claude and Cursor, so if something pops, blame the bots 🤖!
  • All data processing happens locally - nothing goes to servers
  • Since the tool was designed for utility first, there are a few design inconsistencies (alignment & color palette, margins, etc.) that will get fixed on a breezy afternoon
  • This tool is not optimized for mobile devices. Since the data is tabular, it may not display well on phones and is therefore not recommended for mobile use.
